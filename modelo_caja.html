El modelo de caja se compone de cuatro elementos: margin, border, padding y contenido. Contenido del elemento HTML: es todo lo que está dentro del elemento. Este tiene medidas establecidas por las propiedades width y height. Bordes: consiste en el perfil o borde de un elemento. Espaciado interno o padding: consiste en el espacio entre el borde y el contenido del elemento HTML. espaciado externo o margin:el espacio entre el borde y otro elemento. Document
